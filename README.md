# Diane Cruz Portfolio

A personal portfolio website showcasing the skills, projects, and contact information of Diane Cruz, a student developer and UI/UX designer.

---

## Overview
This portfolio website presents Diane Cruz's skills, projects, and contact details in a visually appealing and responsive layout. It highlights web development, UI/UX design expertise, and professional projects for potential employers or collaborators.

---

## Features
- Responsive and modern design with a gradient background.
- Hero section with profile image and introduction.
- About section with personal background and skills.
- Skills section highlighting technical proficiencies with interactive cards.
- Projects section displaying key projects with descriptions and images.
- Contact form with rounded input fields and social media links.
- Smooth scroll animations for sections and elements.
- Mobile-friendly navigation menu with a hamburger toggle.

---

## Tech Stack
- **HTML5** – Structured semantic content
- **CSS3** – Styling, animations, and responsive layouts
- **JavaScript** – Interactivity and scroll animations
- **Google Fonts** – Poppins font for typography
